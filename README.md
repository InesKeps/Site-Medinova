# Site Medinova — Hospital Showcase Website

A responsive showcase website for a hospital. Covers services, departments, team, and contact — structured for easy navigation and professional presentation.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-→%20View%20Site-blue?style=flat)](https://ineskeps.github.io/Site-Medinova)

## Stack

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![SCSS](https://img.shields.io/badge/SCSS-CC6699?style=flat&logo=sass&logoColor=white)

## Features

- Fully responsive — mobile, tablet, desktop
- CSS Grid and Flexbox for page structure
- Media queries for all breakpoints
- Clean, professional medical design
- No external dependencies

## Run locally

```bash
git clone https://github.com/InesKeps/Site-Medinova.git
cd Site-Medinova
open index.html
```
